# 다해드리오 (Daehaderio) - 믿을 수 있는 청소 서비스

다해드리오는 직접 청소와 검증된 업체 중개를 제공하는 플랫폼입니다.

## 🚀 주요 기능

### 🏠 서비스 유형
- **직접 청소**: 다해드리오가 직접 수행하는 프리미엄 청소 서비스
- **업체 연결**: 엄선된 파트너 청소 업체와 고객 연결

### 📋 관리 시스템
- 견적 요청 접수 및 관리
- 실시간 상태 업데이트
- 이메일 알림 시스템
- CSV 내보내기 기능

### 📱 사용자 경험
- 3분 견적 요청
- 모바일 최적화
- 30분 내 연락 보장

## 🛠 기술 스택

- **Frontend**: Next.js 14 (App Router), TypeScript, Tailwind CSS
- **Backend**: Next.js API Routes, Node.js
- **Database**: JSON 파일 기반 (추후 DB 연동 가능)
- **Email**: Resend
- **Deployment**: Vercel
- **Styling**: Tailwind CSS, Lucide Icons

## 📦 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 프로덕션 서버 실행
npm start
```

## 🌐 배포

### Vercel 배포
1. GitHub 저장소 생성
2. Vercel 대시보드에서 저장소 연결
3. 환경 변수 설정
4. 자동 배포 완료

### 환경 변수 설정
```env
RESEND_API_KEY=your_resend_api_key
ADMIN_EMAIL=admin@yourdomain.com
NEXT_PUBLIC_SITE_URL=https://yourdomain.vercel.app
```
